# Neurolabs Product Engineer Challenge

This repository contains my submission for the Neurolabs Product Engineer take-home challenge.

The project implements an end-to-end shelf intelligence pipeline capable of:

- Predicting the SKU of products detected on retail shelf images
- Exposing predictions through an API
- Presenting the results and business insights in a simple dashboard

## Running

```bash
# Create a virtual environment
python -m venv .venv

# Activate it (Windows)
.venv\Scripts\Activate.ps1

# Install dependencies
pip install -r requirements.txt
```

More detailed setup instructions will be added as the project progresses.
